# Blogapp
  A Simple Blog app
# Install mongodb

   https://www.mongodb.com/download-center

  
## Install

```
npm init
```
```
npm install express mongoose body-parser --save
```
```
npm install ejs --save
```
```
npm install method-override -save
```
```
npm install express-saintizer --save
```
## Run
```
mongod
```
```
node app.js
```
```
localhost:5150
```
